1. 레이스 시작
    
    레이스 시작되면 장소를 입력으로 받고, 입력받은 레이스가 시작됩니다.
   

   2. 해당 레이스에 참가하는 드라이버 숫자를 입력받습니다.
   3. 팀정보를 입력합니다.

팀명, 엔진정보, 주행기록을 입력합니다.

총 11개 팀중 화면에 보이는 8팀은 엔진이 정해졌기 때문에  나머지 3개팀은 수기로 입력하여 엔진 정보를 업데이트 합니다.
상위 22명의 드라이버중 3명은 순위권(포디움)이며 다음과 같이 점수를 획득합니다.

참가하는팀 모두 정보를 입력받고 난후 주행기록에 따라 순위가 정해집니다.

한팀의 모든 드라이버가 순위권내 들어갔을 경우 더블 포디움이 되어 점수를 획득합니다.
